# ruth-ibrahim

tiny database.

# usage

```
git clone https://github.com/ohataken/ruth-ibrahim.git
cd ruth-ibrahim
make run
```

# commands

## save command

```
save 7 127
```

this command saves the value 127 at 7th index.

## load command

```
load 7
127
```

this command shows the value 127 at 7th index.
